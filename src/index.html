Formación

A parte del bachillerato poseo el título de Desarrollo de Aplicaciones Web (DAW) terminado en 2020. Desde ese momento empecé mi búsqueda de empleo por diferentes empresas. Al ser año de pandemia, no pude hacer prácticas en empresa. A partir de ahí tuve que formarme por mi cuenta y seguir la búsqueda de empleo

En todo este tiempo, a parte de trabajar en otro sector ( las facturas no se pagaban solas ), hice varios cursos por mi cuenta. Estos son mis conocimientos:

  • Angular y Angular avanzado
  • Javascript, NodeJs(Express)
  • PHP y Laravel
  • HTML5, CSS, Jquery
  • VueJs,ReactJs
  • Java y C#

También he usado Bases de datos MySql, Firebase, MongoDB. Uso también GIT
